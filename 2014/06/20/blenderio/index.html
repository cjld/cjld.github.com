Blender Inside Out

Published

20 June 2014

Tags

  • Blender 12

Contents

Blender Inside Out 对 blender 做了非常系统的介绍，信息量比较大，做些笔记：

Interface

  • N : Property panel
  • T : Tools panel

Selection Tools

  • RMB : select one
  • A : (De)select All / Invert
  • B : Border Select
  • C : Circle Select , SW to change the size
  • Ctrl + LMB : Polygon Select
  • Shift + ... : Deselect
  • MMB : Deselect

Transform Tools

  • G : Translate
  • R : Rotate
  • S : Scale
  • Pivot center for rotate and scale
  • Transform orientation : Global , Local , Normal , Gimbal? , Custom
  • X Y Z Shift Constraint Axis
  • Alt + Space : Select Orientation
  • Shift + D : Duplicate

Using The Search Box To Find Amost Anything

Customizing The Interface Layout

Right click in split line to split or merge

Using The 3D Cursor

  • D + LMB | RMB | Ctrl: draw | erase | line
  • Shift + S : snap menu

Object Mode Versus Edit Mode

  • Tab : switch between two mode
  • Ctrl + Tab : (in Edit Mode) switch select mode {v,e,f}

Mesh Selection Tools

  • L : select link
  • Shift + R : Repeat last command
  • Ctrl + N+ | N- : select more | less around cur select
  • P : separated to another obj
  • Y : split
  • Alt + RMB : select edge loop like — — —
  • Ctrl + Alt + RMB : select edge ring like | | |
  • Ctrl + RMB : select path

Object / Mesh Snapping

  • snap to : volume , face, edge, vertex
  • snap element : center, active, median, closest
  • Ctrl + LMB : duplicate and extrude at 3d cursor
  • Ctrl when transform, enable snap

Edit model

  • E : Extrude and Extrude individual
  • Ctrl + V|F|E : vertex/face/edge menu
  • I : face inset
  • face edge bevel
  • bridge and spin
  • loop cuts
  • knife
  • merge and collapse
  • Mesh Creases for subsdiv
  • mesh display in property to display normal, length, etc.

Modifiers

  • Ctrl + H : hook menu
  • all modifiers Tutorial

Smoothing Groups

  • Mark sharp used by Edge Split modifier
  • Crease used by subsurf modifier

Setting Seams And Unwrapping

  • Mark Seams in Edge menu
  • Ctrl + P : Pack island
  • W : Weld / Align Menu
  • P : Pin, need to enable live unwarp
  • Sculpting UV
  • M move to layer

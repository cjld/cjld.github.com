Unity 在导入模型时点数发生改变

Published

30 March 2014

Tags

  • Unity3D 3

Contents

最近要利用 Unity 做一些关于 mesh 的变形，但是却出现了一些问题，发现原来的 obj 文件有 n 个点，导入到 Unity 中却变多了…

事后找到原因，原来是 Unity 在导入模型时，会把具有不同法向量的点拆分成几个点，因此点数就变多了，解决方法是在导入模型设置里把 Normals 改为 None
